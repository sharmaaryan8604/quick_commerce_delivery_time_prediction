# Quick Commerce Delivery Time Prediction System

An end-to-end Machine Learning web application predicting delivery time across 11 Indian cities for quick-commerce platforms like Blinkit, Instamart and Zepto.

---

## Features

• Ensemble regression model  
• Temporal + spatial feature engineering  
• Label encoding pipeline  
• Feature importance visualization  
• REST prediction API  
• React frontend dashboard  
• Cloud deployment (Render + Vercel)

---

## Tech Stack

Python  
Scikit-learn  
FastAPI  
React  
TailwindCSS  

---

## Model

Random Forest Regression model trained on engineered features including:

• peak-hour flag  
• weekend flag  
• distance-per-item ratio  
• order metadata
